Makefile: designed to compile/clean/run program.
type make to compile the files.
type make run to run the file in terminal.
type make clean to clean the program.

Database_H.h: used to instantiate the methods and variables used within the .cpp file ( also defines namespace ).

Database.cpp: contains the content of the main method, as well as the other methods definitions. temporary variables are used in this file, although all permenant variables are
instantiated in the header file.
